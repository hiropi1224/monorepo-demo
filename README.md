## [公式ドキュメント](https://turbo.build/repo/docs)
### What is a Monorepo?
```
モノレポとは、1つのコードベースに多くの異なるアプリやパッケージを集めたものです。
ポリレポというのは、複数のコードベースを別々に公開し、バージョン管理する方法です。
```
